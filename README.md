# 貓咪咖啡館 V0.51.1

本版新增場景前景遮罩與牆面家具獨立圖層：

- 牆面與地板邊界使用前景遮罩，地面家具超出地板時會自然被牆面遮住。
- 窗邊吊床、拍照背板、靠牆長椅改放在牆面家具圖層，不會被牆面遮罩蓋住。
- 舊存檔中的一般家具會再次依家具高度規則移回安全列。
- 手機直立背景與前景遮罩同步上移，避免圖層錯位。

將 ZIP 解壓縮後，把所有檔案直接部署至 GitHub Pages 根目錄即可。


## V0.51.7 修正
- 手機直式上方資訊列改為雙列配置，避免按鈕與文字擠成一坨。
- 調整地板四角座標與可放置邏輯，減少大面積空地無法擺放。
- 放寬家具安全範圍，降低隱形碰撞造成的死區。
- 將部分新增家具改為原創點陣感圖像，維持類開羅風味但不直接使用他人素材。


## V0.51.8 統一地板系統
- 地板顯示、格線、吸附、碰撞使用同一四角投影。
- Canvas 重繪地板表面，覆蓋舊背景的重複斜線。
- 地毯移到獨立地板圖層，避免被家具切片與前景遮罩誤裁。
- 擴張左下方地板多邊形，補齊可見地板的擺放範圍。


## V0.51.9 修正
- 完整撤回可視地板覆蓋層。
- 恢復原場景背景比例，不再拉伸。
- 地板幾何只用於透明碰撞與吸附。
- 格線僅在格線開啟或家具佈置時顯示。
- 地毯圖層取消裁切。


## V0.52.0
