# 貓咪咖啡館 V0.51.1

本版新增場景前景遮罩與牆面家具獨立圖層：

- 牆面與地板邊界使用前景遮罩，地面家具超出地板時會自然被牆面遮住。
- 窗邊吊床、拍照背板、靠牆長椅改放在牆面家具圖層，不會被牆面遮罩蓋住。
- 舊存檔中的一般家具會再次依家具高度規則移回安全列。
- 手機直立背景與前景遮罩同步上移，避免圖層錯位。

將 ZIP 解壓縮後，把所有檔案直接部署至 GitHub Pages 根目錄即可。
